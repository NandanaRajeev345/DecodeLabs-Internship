# Project 2: Data Classification using KNN

## 📌 Project Description
This project is a supervised machine learning model that classifies Iris flowers into three species (Setosa, Versicolor, Virginica) based on their features such as sepal length, sepal width, petal length, and petal width.

The model uses the K-Nearest Neighbors (KNN) algorithm to learn patterns from data and make predictions.

---

## 📊 Dataset Used
- Iris dataset from `scikit-learn`

---

## 🤖 Algorithm Used
- K-Nearest Neighbors (KNN)
